# Developer Foundations

Este repositório documenta minha formação técnica em desenvolvimento de software.

Ele foi estruturado como uma linha do tempo de aprendizado, com foco na construção de fundamentos sólidos, organização profissional e boas práticas de versionamento desde o início.

O objetivo é registrar, de forma clara e progressiva, minha evolução técnica na área de tecnologia.

## Estrutura

Os conteúdos estão organizados em módulos numerados, representando etapas do aprendizado:

- `00-introducao` — Contexto do projeto, objetivos e metodologia  
- `01-fundamentos` — Fundamentos essenciais (Git/GitHub e lógica de programação)  
- `02-front-end` — Desenvolvimento front-end *(planejado)*  
- `03-back-end` — Desenvolvimento back-end *(planejado)*  
- `04-banco-de-dados` — Bancos de dados *(planejado)*  
- `05-projetos` — Projetos práticos *(planejado)*

## Metodologia

- Aprendizado incremental  
- Documentação contínua  
- Versionamento frequente com Git  
- Organização alinhada a padrões profissionais
