# Developer Foundations

Este repositório documenta minha formação técnica em desenvolvimento de software.

Ele está estruturado como uma linha do tempo de aprendizado, com foco na construção de bases sólidas, organização profissional e boas práticas de versionamento desde o início.

O objetivo é registrar de forma clara e progressiva minha evolução técnica na área de tecnologia.

### Estrutura

O conteúdo está organizado em módulos numerados, representando etapas de aprendizado:

- `00-introduction` — Contexto do projeto, objetivos e metodologia  
- `01-fundamentals` — Fundamentos principais (Git/GitHub e lógica de programação)  
- `02-front-end` — Desenvolvimento front-end *(planejado)*  
- `03-back-end` — Desenvolvimento back-end *(planejado)*  
- `04-databases` — Bancos de dados *(planejado)*  
- `05-projects` — Projetos práticos *(planejado)*  

### Metodologia

- Aprendizado estruturado  
- Documentação contínua  
- Versionamento frequente com Git  
- Organização alinhada a padrões profissionais
