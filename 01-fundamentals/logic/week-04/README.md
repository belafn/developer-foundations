# Semana 04 — Funções, Escopo e Estruturação de Lógica

Esta semana marca a transição de scripts sequenciais para uma abordagem mais estruturada e modular da programação.

O foco principal é compreender profundamente funções, escopo de variáveis, closures e a aplicação desses conceitos na resolução de problemas mais complexos.

A ênfase deixa de ser apenas executar código e passa a ser organizar lógica de forma reutilizável, previsível e profissional.

## Tópicos abordados

### 01. Fundamentos de funções
- Declaração de funções
- Invocação (chamada de função)
- Separação de responsabilidades
- Reutilização de código

### 02. Parâmetros e retorno
- Parâmetros formais e argumentos
- Valores de retorno (`return`)
- Fluxo de execução dentro de funções
- Diferença entre função que executa ação e função que retorna valor

### 03. Escopo e closures
- Escopo global
- Escopo de função
- Escopo de bloco
- Encapsulamento
- Conceito de closure
- Persistência de estado através de funções

### 04. Arrow functions
- Sintaxe moderna (ES6+)
- Diferença entre function declaration e arrow function
- Uso implícito de return
- Boas práticas de legibilidade

### 05. Resolução de problemas
- Estruturação de lógica em etapas
- Decomposição de problemas
- Criação de funções auxiliares
- Pensamento algorítmico

### 06. Mini projeto
Aplicação integrada dos conceitos estudados:
- Organização de código
- Separação de responsabilidades
- Reutilização de funções
- Clareza estrutural

## Objetivos de aprendizado

- Entender funções como unidade básica de organização de código
- Dominar passagem de parâmetros e retorno de valores
- Compreender como o escopo afeta o comportamento do programa
- Entender closures como mecanismo de encapsulamento
- Aplicar funções na resolução estruturada de problemas
- Desenvolver raciocínio modular
- Aproximar o código de padrões utilizados no mercado

## Observações

Esta semana representa um avanço estrutural importante.

A partir daqui, o código deixa de ser apenas sequencial e passa a ser modular, reutilizável e mais próximo de aplicações reais.

O mini projeto consolida a transição entre aprendizado guiado e construção autônoma de lógica.

# Week 04 — Functions, Scope and Structured Logic

This week represents a transition from sequential scripts to a more structured and modular programming approach.

The main focus is to deeply understand functions, variable scope, closures, and how to apply these concepts to more complex problem-solving.

The emphasis shifts from simply executing code to organizing logic in a reusable and professional way.

## Topics covered

### 01. Function fundamentals
- Function declaration
- Function invocation
- Separation of concerns
- Code reuse

### 02. Parameters and return
- Formal parameters and arguments
- Return values (`return`)
- Execution flow inside functions
- Difference between performing an action and returning a value

### 03. Scope and closures
- Global scope
- Function scope
- Block scope
- Encapsulation
- Closure concept
- State persistence through functions

### 04. Arrow functions
- Modern ES6+ syntax
- Differences between traditional functions and arrow functions
- Implicit return
- Readability best practices

### 05. Problem solving
- Logical decomposition
- Step-by-step reasoning
- Helper functions
- Algorithmic thinking

### 06. Mini project
Integrated application of the studied concepts:
- Code organization
- Separation of responsibilities
- Function reuse
- Structural clarity

## Learning goals

- Understand functions as the core unit of code organization
- Master parameter passing and return values
- Understand how scope impacts program behavior
- Grasp closures as an encapsulation mechanism
- Apply structured problem-solving
- Develop modular thinking
- Move closer to real-world coding patterns

## Notes

This week represents an important structural shift.

From this point forward, code becomes more modular, reusable, and aligned with professional development practices.
