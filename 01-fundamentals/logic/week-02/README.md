# Semana 02 — Fundamentos de JavaScript (Parte II)

Esta semana é focada em fluxo de controle e tomada de decisões em JavaScript.

## Tópicos abordados
- Operadores de comparação
- Operadores lógicos
- Estruturas condicionais (if / else)
- Estrutura switch
- Exercícios práticos

## Objetivos de aprendizado
- Entender como programas tomam decisões
- Combinar múltiplas condições de forma segura
- Escrever lógicas mais claras e legíveis
- Praticar resolução de problemas usando condicionais

## Observações
As práticas de depuração são aplicadas ao longo dos exercícios utilizando `console.log`
e raciocínio passo a passo
