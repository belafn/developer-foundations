# Semana 06 — Funções Avançadas, Imutabilidade e Organização de Código

Esta semana representa um salto de maturidade técnica no projeto.

O foco deixa de ser apenas escrever código funcional e passa a incluir qualidade estrutural, previsibilidade e organização profissional.

Introduzimos conceitos fundamentais para desenvolvimento moderno em JavaScript, como funções de alta ordem, métodos avançados de arrays, imutabilidade e modularização.

Além disso, começamos a aplicar princípios de Clean Code na prática.

## Tópicos abordados

### 01. Funções avançadas
- Funções como valores (first-class citizens)
- Funções de alta ordem (higher-order functions)
- Callbacks
- Composição de funções
- Encadeamento de responsabilidades

### 02. Métodos avançados de arrays
- `map`
- `filter`
- `reduce`
- `some`
- `every`
- `find`
- Encadeamento de métodos (method chaining)
- Transformação e agregação de dados

### 03. Imutabilidade
- Diferença entre mutação e imutação
- Cópias superficiais (shallow copy)
- Uso de spread operator para evitar mutação
- Previsibilidade de estado
- Boas práticas para evitar efeitos colaterais

### 04. Introdução a módulos
- Separação de responsabilidades em arquivos
- Conceito de modularização
- Organização de código em partes reutilizáveis
- Preparação para uso de `import` e `export`

### 05. Princípios de Clean Code
- Nomeação clara e semântica
- Funções pequenas e com responsabilidade única
- Evitar duplicação de código
- Legibilidade como prioridade
- Estruturação lógica de arquivos

### 06. Mini projeto
Aplicação integrada dos conceitos:
- Manipulação avançada de arrays
- Uso de funções de alta ordem
- Aplicação de imutabilidade
- Organização modular do código
- Aplicação prática de princípios de Clean Code

## Objetivos de aprendizado

- Entender funções como unidades reutilizáveis e composáveis
- Dominar métodos modernos de manipulação de arrays
- Desenvolver código previsível através de imutabilidade
- Organizar projetos em módulos
- Aplicar princípios de código limpo
- Aproximar o desenvolvimento de padrões utilizados no mercado

## Evolução técnica até aqui

- Week 01–03: Fundamentos, controle de fluxo e estruturas básicas
- Week 04: Estruturação com funções e escopo
- Week 05: Modelagem de dados com objetos
- Week 06: Código mais funcional, previsível e organizado

Esta semana marca a transição do aprendizado sintático para pensamento arquitetural básico.

# Week 06 — Advanced Functions, Immutability and Code Organization

This week represents a significant technical maturity step in the project.

The focus moves beyond writing working code to building structured, predictable, and professionally organized solutions.

We introduce key modern JavaScript concepts such as higher-order functions, advanced array methods, immutability, and modularization.

Clean Code principles are also applied in practice.

## Topics covered

### 01. Advanced functions
- Functions as first-class citizens
- Higher-order functions
- Callbacks
- Function composition
- Responsibility chaining

### 02. Advanced array methods
- `map`
- `filter`
- `reduce`
- `some`
- `every`
- `find`
- Method chaining
- Data transformation and aggregation

### 03. Immutability
- Mutation vs immutability
- Shallow copies
- Spread operator usage
- State predictability
- Avoiding side effects

### 04. Introduction to modules
- Separation of concerns into files
- Modular code organization
- Reusable structures
- Preparation for `import` and `export`

### 05. Clean Code principles
- Clear and semantic naming
- Small, single-responsibility functions
- Avoiding duplication
- Readability as a priority
- Logical file structure

### 06. Mini project
Integrated application:
- Advanced array manipulation
- Higher-order functions
- Immutability practices
- Modular structure
- Practical Clean Code application

## Learning goals

- Apply clean code practices
- Move closer to real-world development standards
