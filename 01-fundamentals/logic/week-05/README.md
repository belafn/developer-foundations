# Semana 05 — Objetos e Modelagem de Dados

Esta semana aprofunda a estruturação de dados em JavaScript através do uso de objetos.

O foco deixa de ser apenas lógica procedural e passa a incluir organização de informação, modelagem de entidades e manipulação avançada de estruturas compostas.

Aqui começa a transição para um pensamento mais próximo do desenvolvimento de aplicações reais, onde dados estruturados são essenciais.

## Tópicos abordados

### 01. Fundamentos de objetos
- Criação de objetos literais
- Propriedades e métodos
- Acesso via notação de ponto e colchetes
- Atualização e adição de propriedades
- Iteração sobre objetos

### 02. Objetos aninhados (Nested Objects)
- Estruturas hierárquicas
- Acesso a propriedades profundas
- Manipulação de dados estruturados
- Organização de entidades complexas

### 03. Destructuring
- Desestruturação de objetos
- Extração de propriedades
- Renomeação de variáveis
- Destructuring em parâmetros de função
- Melhoria de legibilidade

### 04. Spread e Rest
- Operador spread (`...`) em objetos
- Cópias superficiais (shallow copy)
- Imutabilidade básica
- Operador rest em objetos
- Manipulação dinâmica de propriedades

### 05. Modelagem de dados
- Representação de entidades do mundo real
- Estruturação coerente de informações
- Organização de atributos
- Separação de responsabilidades entre dados
- Pensamento orientado a domínio

### 06. Mini projeto
Aplicação integrada dos conceitos estudados:
- Criação de múltiplos objetos relacionados
- Manipulação de dados estruturados
- Uso de destructuring e spread
- Organização lógica de entidades

## Objetivos de aprendizado

- Entender objetos como estrutura central de organização de dados
- Modelar entidades de forma clara e coerente
- Manipular estruturas aninhadas com segurança
- Aplicar destructuring para escrever código mais limpo
- Utilizar spread e rest para trabalhar com imutabilidade
- Evoluir do pensamento baseado apenas em variáveis para pensamento baseado em entidades

## Observações

Esta semana representa um avanço importante na maturidade técnica.

A partir daqui, o foco não está apenas em executar lógica, mas em estruturar dados de maneira profissional — habilidade essencial para trabalhar com APIs, bancos de dados e aplicações reais.

O mini projeto consolida a prática de modelagem e manipulação de objetos, aproximando o código de padrões utilizados no mercado.

# Week 05 — Objects and Data Modeling

This week deepens data structuring in JavaScript through the use of objects.

The focus shifts from purely procedural logic to information organization, entity modeling, and advanced manipulation of composite structures.

This marks the beginning of a mindset closer to real-world application development, where structured data is essential.

## Topics covered

### 01. Object fundamentals
- Object literals
- Properties and methods
- Dot and bracket notation
- Updating and adding properties
- Object iteration

### 02. Nested objects
- Hierarchical structures
- Accessing deep properties
- Manipulating structured data
- Organizing complex entities

### 03. Destructuring
- Object destructuring
- Property extraction
- Variable renaming
- Destructuring in function parameters
- Readability improvements

### 04. Spread and rest
- Spread operator (`...`) with objects
- Shallow copies
- Basic immutability concepts
- Rest operator in objects
- Dynamic property handling

### 05. Data modeling
- Representing real-world entities
- Structuring information coherently
- Attribute organization
- Separation of data responsibilities
- Domain-oriented thinking

### 06. Mini project
Integrated application of studied concepts:
- Creating related objects
- Manipulating structured data
- Using destructuring and spread
- Logical entity organization

## Learning goals

- Understand objects as the core data structure in JavaScript
- Model entities clearly and coherently
- Safely manipulate nested structures
- Use destructuring for cleaner code
- Apply spread and rest for better data handling
- Transition from variable-based thinking to entity-based thinking

## Notes

This week represents a significant technical maturity step.

From this point forward, the focus expands beyond logic execution to professional data structuring — a fundamental skill for working with APIs, databases, and real-world applications.
