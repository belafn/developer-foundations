# Semana 03 — Fundamentos de JavaScript (Parte III)

Esta semana aprofunda conceitos fundamentais da linguagem JavaScript, com foco em estruturas de repetição e manipulação básica de dados.

O objetivo é combinar controle de fluxo com arrays e strings, avançando da prática guiada para a aplicação prática de conceitos.

## Tópicos abordados
- Estruturas de repetição (loops)
- Laços `for`, `while` e `do...while`
- Arrays: criação e acesso a dados
- Métodos básicos de arrays
- Métodos comuns de strings
- Exercícios práticos
- Simulações de cenários reais

## Objetivos de aprendizado
- Iterar dados utilizando diferentes tipos de loops
- Compreender quando utilizar cada estrutura de repetição
- Manipular arrays e strings de forma segura
- Aplicar métodos de arrays e strings na resolução de problemas
- Combinar múltiplos conceitos para construir lógicas estruturadas
- Evoluir do treino guiado para a aplicação prática

## Observações
Os exercícios práticos focam na fixação de conceitos isolados, enquanto as simulações
exploram cenários mais próximos de problemas reais, exigindo a combinação de diferentes
estruturas e raciocínio lógico.

# Week 03 — JavaScript Fundamentals (Part III)

This week deepens core JavaScript concepts, focusing on loop structures and basic data manipulation.

The goal is to combine control flow with arrays and strings, progressing from guided practice to applied problem-solving.

## Topics covered
- Loop structures
- `for`, `while`, and `do...while` loops
- Arrays: creation and data access
- Basic array methods
- Common string methods
- Practice exercises
- Real-world scenario simulations

## Learning goals
- Iterate over data using different loop strategies
- Understand when to use each type of loop
- Manipulate arrays and strings safely
- Apply array and string methods to solve problems
- Combine multiple concepts to build structured logic
- Transition from guided practice to applied problem-solving

## Notes
Practice exercises focus on reinforcing individual concepts, while simulations introduce
more realistic scenarios that require combining multiple structures and logical reasoning.
