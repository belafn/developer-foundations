# Semana 01 — Fundamentos de JavaScript

Esse diretório contém minha primeira semana de prática com JavaScript, com foco nos conceitos centrais da linguagem e na interação básica com o ambiente de execução.

O objetivo desta semana foi desenvolver familiaridade com a sintaxe, o fluxo de execução e técnicas fundamentais de depuração.

## Tópicos abordados

### 01. Conceitos básicos do console
Introdução ao uso do console para exibição de resultados, entendendo como o JavaScript comunica informações durante a execução.

### 02. Variáveis
Estudo da declaração de variáveis, atribuição de valores e armazenamento básico de dados em JavaScript.

### 03. Template strings
Uso de template literals para criar strings dinâmicas e melhorar a legibilidade do código.

### 04. Simulação
Simulações simples para praticar raciocínio lógico, interação entre variáveis e execução sequencial.

### 05. Depuração
Técnicas básicas de debugging, incluindo inspeção de valores, acompanhamento do fluxo de execução e identificação de erros.

## Observações

Todos os arquivos deste diretório foram criados com fins educacionais e representam o progresso gradual ao longo da minha primeira semana de estudos.
